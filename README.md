CAS Webapp Template
============================

## usage

```xml
<dependency>
    <groupId>com.github.cas-side</groupId>
    <artifactId>cas-webapp-template</artifactId>
    <version>5.3.11</version>
</dependency>
```

## change list

- 默认 http，没有使用 https，端口: 8080
